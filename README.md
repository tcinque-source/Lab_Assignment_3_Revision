# Lab_Assignment_3_Revision
Functional code for lab assignment 3 which will compile and perform required tasks.

When the program is run the terminal will be blank and will recieve text input from the user. When the user enters their paragraph the program will return the total number of words and total sentences of the paragraph. Following this, the user will be prompted to enter a target word, which the program will find, and a count of the total number of times the target word is used will be returned.
